# Release Notes

## v0.1.0 (2026-03-16)

Initial release of identity-skills, forked from [dotnet-skills](https://github.com/Aaronontheweb/dotnet-skills) v1.3.0 and adapted for Duende IdentityServer development.

### What Changed

- Removed 16 irrelevant skills (Akka.NET, Mailpit, MJML, Testcontainers, serialization, slopwatch, ilspy-decompile, marketplace-publishing, skills-index-snippets, playwright-ci-caching)
- Removed 3 irrelevant agents (Akka.NET specialist, DocFX specialist, Roslyn incremental generator specialist)
- Added 10 placeholder skills for IdentityServer, OAuth/OIDC, BFF, token management, claims, ASP.NET Core authentication and authorization, and identity security hardening
- Added 2 new agents: `identity-server-specialist`, `oauth-oidc-specialist`
- Rebranded all metadata for DuendeSoftware/identity-skills
- Updated scripts for new skill category prefixes (identityserver-*, duende-*, aspnetcore-*, identity-*, token-*, claims-*, oauth-*)
- 5 surviving general-purpose skills annotated with identity-specific TODO adaptation guidance
